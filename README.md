# Porsche Taycan - Site Vitrine 🏎️⚡

Site vitrine personnel dédié à la Porsche Taycan, première voiture 100% électrique de la marque allemande.

## 📋 Description

Ce projet personnel présente la Porsche Taycan à travers son histoire, ses différents modèles, ses performances et ses records. Créé pour partager ma passion pour l'automobile électrique et mettre en pratique mes compétences en développement web.

## 🎯 Objectifs du projet

- Partager ma passion pour la Porsche Taycan
- Créer un site vitrine moderne et responsive
- Expérimenter avec des animations CSS et des effets visuels
- Présenter des informations de manière claire et attractive

## ✨ Fonctionnalités

- **Navigation fluide** : Menu fixe avec navigation par ancres (scroll smooth)
- **Timeline interactive** : Historique de la Taycan avec effet hover
- **Cartes de modèles** : 3 modèles principaux avec effet de zoom au survol
- **Graphique de ventes** : Visualisation des données avec Chart.js
- **Design moderne** : Palette de couleurs Porsche (rouge #E10600 et gris foncé)
- **Responsive** : Adapté mobile, tablette et desktop

## 🛠️ Technologies utilisées

- **HTML5** : Structure sémantique
- **CSS3** : Flexbox, animations, transitions
- **JavaScript** : Chart.js pour les graphiques
- **Font** : Michroma (Google Fonts)

## 🚀 Installation et utilisation

1. Cloner le repository
```bash
git clone https://github.com/ton-username/porsche-taycan.git
```

2. Ouvrir le fichier `index.html` dans un navigateur

Aucune installation de dépendances nécessaire, le projet utilise des CDN pour Chart.js.
